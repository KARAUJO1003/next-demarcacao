### Como clonar o projeto:

1 - Primeiro:, instalação de dependências:

```bash
npm install
# or
yarn install

```
2 - Segundo: Inicie o projeto com o comando:

```bash
npm run dev
# or
yarn dev

```

Abra o navegador no endereço [http://localhost:3000](http://localhost:3000) .
